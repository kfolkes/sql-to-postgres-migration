# Security Assessment Guide

This directory contains security baselines, hardening guides, and scan results for the SQL Server to PostgreSQL migration.

## Structure

- `source-baseline.md` - SQL Server security posture snapshot
- `target-hardening.md` - PostgreSQL security configuration guide
- `defender-config.md` - Microsoft Defender for Open-Source DBs setup
- `sec-check-results/` - sec-check scan outputs (before/after)
